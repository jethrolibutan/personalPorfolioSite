Contractor Control (Class Project)
ReactJs, Django, PostgreSQL

My team and I developed a full stack web application designed specifically for contractors to streamline their inventory management, quote generation, invoicing, and time tracking for a school project. This project comprises of ten fully responsive pages, each developed using React components built with React hooks, ensuring a smooth user experience. The backend of the application is powered by a Django REST API, which seamlessly handles data management and business logic. For secure user management, I implemented Django’s JWT authentication, allowing users to safely create accounts, log in, and access personalized features. The application uses PostgreSQL as its database. This project showcases my ability to integrate front-end and back-end technologies to create a cohesive and functional web application that meets real-world business needs.

farm2clinic.com
ReactJs, HTML, CSS

I learned about Farm2Clinic at the beginning of last Fall Semester last year when I took a community engaged-participatory-research honors class led by Dr. Sastre. I learned that Farm2Clinic had no digital presence after meeting with Dr. Sastre one-on-one. A couple of meetings later she agreed to let me build the website. I used React.js, MaterialUI, and JavaScript. React.js is a JavaScript library used for building reusable UI components and user interfaces. It lets developers compose UIs (user interfaces) from small and isolated and reusable React components. I also specifically used a React.js Library known as Material UI. Material UI is very useful because it is an open-source React component library that includes many prebuilt components that allows developers to create web applications efficiently. After a month of weekly meetings with Dr. Sastre, teaching myself a new JavaScript framework (React.js) and its various fundamentals (from JSX to what “props” are), and many commits to GitHub later I managed to get a working version of the website online! The website is now up and running and the link to the website itself is: farm2clinic.com. To host the website on the internet, the domain “farm2clinic.com” was bought from GoDaddy.com and I deployed the website to “farm2clinic.com” using Netlify.com.
